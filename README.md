# Build from source

## Prerequisite
- node
- npm or yarn

1. Clone repository
```bash
git clone https://github.com/glendell03/dijkstra-algorithm-mapbox.git
```
2. Cd into the folder
3. Install dependencies
```bash
npm i
```
4. Start application
```bash
npm start
```
